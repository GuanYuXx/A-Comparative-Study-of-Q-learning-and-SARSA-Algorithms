# Specs: Cliff Walking — Q-Learning vs SARSA

## 環境規格
- 網格大小：4 × 12
- 起點 (S)：(3, 0) — 左下角
- 終點 (G)：(3, 11) — 右下角
- 懸崖：(3, 1) ~ (3, 10)
- 動作空間：上 (0)、下 (1)、左 (2)、右 (3)
- 狀態空間：48 個狀態

## 獎勵規格
| 事件 | 獎勵 |
|------|------|
| 正常移動 | -1 |
| 掉落懸崖 | -100，回到起點 |
| 到達終點 | -1，結束回合 |

## 演算法規格

### Q-Learning (Off-policy)
- 更新：Q(s,a) ← Q(s,a) + α[r + γ·maxQ(s',a') − Q(s,a)]
- 特性：學習最優策略，與探索策略無關

### SARSA (On-policy)
- 更新：Q(s,a) ← Q(s,a) + α[r + γ·Q(s',a') − Q(s,a)]
- 特性：學習與探索策略一致的策略，較保守

## 超參數規格
| 參數 | 數值 |
|------|------|
| Episodes (回合數) | 500 |
| 學習率 α | 0.1 |
| 折扣因子 γ | 0.9 |
| 探索率 ε | 0.1 |
